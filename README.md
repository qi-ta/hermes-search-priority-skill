## 📖 What Is This?

A **search priority specification** for Hermes Agent that tells the AI **where** to look and **how** to verify information.

### 🎯 The Problem
AI agents default to low-quality search results — SEO spam, outdated docs, marketing fluff.

### ✅ The Solution
A 5-tier evidence hierarchy + domain-specific source maps that route every query to **the most authoritative source first**.

## 📊 Evidence Level System

| Level | Description | Examples |
|:---:|:---|---:|
| **L1** 🥇 | Primary Sources | Official GitHub, HuggingFace, arXiv |
| **L2** 🥇 | Creator First-Hand | Author X/Twitter, Official Blog, Release Notes |
| **L3** 🥈 | Reproducible Content | Runnable code, demos, benchmarks |
| **L4** 🥉 | Community Validation | Reddit, Hacker News |
| **L5** 🏁 | Media Relays | Tech news, marketing (leads only) |
